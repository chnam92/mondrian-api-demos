🗺️ Google Maps API 프록시 테스트

⚙️ 설정

🗺️ Google Maps JavaScript API

프록시를 통해 Google Maps JavaScript API를 로드하고 지도를 표시합니다.

🏢 Places Library (클라이언트 사이드)

Places Library를 사용한 실제 Autocomplete 위젯과 PlacesService 예제입니다.

📍 Autocomplete Widget

실제 Google Places Autocomplete를 사용한 주소 입력 필드입니다.

🔍 PlacesService Search

PlacesService를 사용한 텍스트 검색과 근처 장소 검색입니다.

📍 지오코딩 API

주소를 좌표로 변환하거나 좌표를 주소로 변환합니다.
/google/wsapi/maps/api/geocode/json
순방향 지오코딩 (주소 → 좌표)
역방향 지오코딩 (좌표 → 주소)

🔍 Places Autocomplete

사용자 입력에 따른 장소 자동완성 기능을 테스트합니다.
/google/wsapi/maps/api/place/autocomplete/json

📋 Places Details

특정 장소의 상세 정보를 가져옵니다.
/google/jsapi/maps/api/place/details/json

🔎 Places Text Search

텍스트 기반으로 장소를 검색합니다.
/google/jsapi/maps/api/place/textsearch/json

📍 Places Nearby Search

특정 위치 근처의 장소를 검색합니다.
/google/jsapi/maps/api/place/nearbysearch/json

🎯 Find Place

텍스트 또는 전화번호로 장소를 찾습니다.
/google/jsapi/maps/api/place/findplacefromtext/json

🛣️ Directions

출발지와 목적지 간의 경로를 검색합니다.
/google/jsapi/maps/api/directions/json

📏 Distance Matrix

여러 출발지와 목적지 간의 거리와 시간을 계산합니다.
/google/jsapi/maps/api/distancematrix/json

📷 Places Photos

장소의 사진을 가져옵니다. (이미지 스트리밍)
/google/wsapi/maps/api/place/photo

🆕 New Places API (v1) - 2024/2025

최신 Google Places API v1을 사용한 POST 요청 기반 API 테스트

🔍 Text Search (New)

새로운 Places API v1 텍스트 검색 - POST 요청 방식
POST /v1/places:searchText

📍 Nearby Search (New)

새로운 Places API v1 근처 검색 - 위치 기반 POST 요청
POST /v1/places:searchNearby

🔍 Autocomplete (New)

새로운 Places API v1 자동완성 - POST 요청 방식
POST /v1/places:autocomplete

📋 Place Details (New)

새로운 Places API v1 장소 상세정보 - GET 요청
GET /v1/places/{placeId}

📷 Place Photos (New)

새로운 Places API v1 사진 - GET 요청 (resource name 사용)
GET /v1/places/{placeId}/photos/{photoReference}/media

🛣️ New Routes API (v2) - 2024/2025

최신 Google Routes API v2를 사용한 POST 요청 기반 경로/거리 계산 API 테스트

🛣️ Compute Route (New)

새로운 Routes API v2 경로 계산 - POST 요청 방식 (JSON 기반)
POST /directions/v2:computeRoutes

📊 Compute Route Matrix (New)

새로운 Routes API v2 거리 매트릭스 - POST 요청 방식 (다중 출발지/목적지)
POST /distanceMatrix/v2:computeRouteMatrix
